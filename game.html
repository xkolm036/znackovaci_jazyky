Úvod

Jako téma semestrálního projektu jsem si vybral nastavení hry. Toto téma jsem si zvolil protože si myslím, že v tomto směru se v praxi XML používá nejčasteji.

XML soubor slouží ke konfiguraci dané hry. Pomocí XML lze nastavit kvalitu grafického zobrazení, kvalitu zvuku a obecné informace o uživateli.

Použité nástroje

Pro tvorbu daného programu jsem využil Microsoft Visual Studio a internetový prohlížeč Mozilla Firefox, dále jsem využil stránky W3.org a w3Schools.

W3.org jsem zároveň požil jakož to oficiální validátor.

Popis transformace

Pomocí transformace jsem z XML souboru vybral elementy, které obsahují onformace o hráči, dále jsem pomoci cysklu for-each vybral veškeré grafické parametry, jejiž hodnota je High nebo Ultra. Nakonec jsem tato vybraná data transformoval do HTML.

Závěr

Při tvorbě této práce jsem se naučil jak spravně pracovat s XML a jak provádět jeho transformaci.
Pokud by se jednalo o reálnou aplikaci by bylo nutné u každého druhu elementu zvlášť definovat omezení a logiku, co kdy muže nebo nesmí být povoleno. Dále by bylo nutné zabezpečit všechna citlivá data.
